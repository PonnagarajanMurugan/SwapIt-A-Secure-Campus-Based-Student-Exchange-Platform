# SwapIt-A-Secure-Campus-Based-Student-Exchange-Platform


SwapIt is a dedicated digital marketplace created for students to conveniently buy, sell, and exchange products within their university network, ensuring trustworthy interactions and a smooth trading process.

# About

SwapIt is a protected and easy-to-use online trading platform built specifically for university students. It allows members of the campus community to exchange, purchase, and sell items among themselves in a reliable environment. Unlike informal social media groups, this system provides a restricted-access platform limited to verified students, which increases safety and credibility. Users can upload product listings with photos and descriptions, while others can explore items using search and filter options. A secure login and verification mechanism ensures that only genuine users can enter the platform. The built-in chat feature helps buyers and sellers communicate efficiently. By encouraging reuse and resale of items, SwapIt supports eco-friendly practices and helps minimize waste on campus. Overall, it delivers a structured and dependable space for student-to-student commerce.

# Features

- Offers a verified student-only trading platform.

- Allows users to quickly upload, discover, and filter product listings.

- Implements Firebase Authentication for secure and trusted login access.

- Enables product photo uploads through Firebase Storage.

- Provides a clean and simple interface for smooth buying and selling activities.

# Requirements

- Operating System: Windows 10/11 or Linux recommended for development and backend execution.

- Development Setup: Node.js (version 16 or above) and npm required for server-side setup and package management.

- Frontend Technologies: HTML5, CSS3, JavaScript, and Tailwind CSS to design a responsive and modern interface.

- Backend & Database: Express.js to build APIs and MongoDB to manage user and product data.

- Security & Authentication: Firebase Authentication for secure login, verification, and identity control.

- Media Storage: Firebase Storage for handling and serving uploaded item images.

- IDE/Tools: Visual Studio Code for coding, testing, and debugging.

- Version Control: Git and GitHub for tracking changes and collaboration.

- Libraries & Packages: Tools such as Mongoose, Multer (if required), dotenv, and Firebase Admin SDK for backend integration.

# Results and Impact

SwapIt greatly enhances the way students trade goods within the campus by delivering a protected, quick, and intuitive marketplace system. With verified access, straightforward item posting, and well-organized browsing features, the platform makes student trading easier and more reliable.

The project improves access to low-cost resources, encourages reuse of products, and strengthens student collaboration. SwapIt also lays the groundwork for expanding digital campus services and helps build a more connected and resource-efficient student ecosystem.
